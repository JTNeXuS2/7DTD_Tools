### request Mods

Enable in webapi in LSTY_Data\appsettings.json

Enable Chat log in Mods\ServerTools_Config\ServerToolsConfig.xml (no longer 2.0+, replacet via telnet method )
    ```<Tool Name="Chat_Logger" Enable="True" />```
